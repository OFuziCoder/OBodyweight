Version 1.3.0

- Body archetypes: each NPC now gets one of 15 coherent body types (Balanced, Slim, Rectangle,
  Pear, Top-heavy, Hourglass, Voluptuous, Apple/Soft, BBW, Athletic, Athletic-curvy, Obese,
  Stocky, Petite, Amazon) that drives bust/waist/hips/belly/tone together. Replaces the old
  independent trait rolls as the primary shape driver (light traits still add micro-variation),
  so bodies read as intentional, recognizable shapes instead of random combinations.
- Average-centered distribution: the MEDIAN body is now balanced/average in every measure, with
  petite and curvy bodies tapering off to the sides. The old distribution pushed most NPCs to the
  extremes (the "everyone is a prominent hourglass" look) — now extremes are the minority.
- Obesity & more body variety: BBW (big soft curvy) and Obese archetypes added, plus Rectangle
  (straight/boyish), Athletic-curvy and Stocky.
- Per-body physics (CBPC, optional): physics now scales with the actual body. Bounce amplitude and
  collider size follow the body's size and softness (bigger/softer = more jiggle + larger colliders;
  toned/lean = firmer/tighter). Uses CBPC's interpolation API per NPC; soft dependency — no CBPC, no
  change. Drop-in config files included (CBPCBounceinterpolationconfig_OBW / CBPCCollisioninterpolationconfig_OBW).
- Body-type API for other mods: OBW_Native.GetArchetypeId(actor) (0-14) and GetArchetypeName(actor)
  ("Pear", "BBW", "Hourglass"...) let other plugins read an NPC's generated body type.
- MCM "Reprocess all loaded NPCs" button: re-applies the current generation + physics to every NPC
  around you without waiting for cell reloads — handy after changing settings or tuning.
- Smoother vertices retained: the soft-cap that keeps big bodies from spiking the sculpted mesh
  still applies on top of the archetype deltas (no breaks/spikes).

Version 1.2.1

- Fix: the re-roll hotkey rebind (MCM) now takes effect immediately. Assigning a new key
  re-registers it directly on the handler, so the new key works (and the old one stops)
  right away, with no reload needed.

Version 1.2.0

- Seam-free: OBW no longer changes an NPC's real weight value (that caused neck seams and
  body-vs-outfit weight mismatches in some setups). All body-size variety now comes from
  morphs, so head, body and outfit always match. The face keeps the NPC's editor weight.
- VR support: the re-roll hotkey now works in VR — it targets the NPC you're gazing at (an
  HMD cone-cast), falling back to yourself when nothing is in view. Desktop is unchanged
  (normal crosshair); the cone-cast only runs in VR.
- Male body tuning: tamed muscle/mass so men aren't over-built, plus a proportionality
  binding so every part scales together (no big arms on a thin chest).
- Male build slider (MCM): scale overall male corpulence (0-2x), proportions preserved.
- Performance: fewer body rebuilds per NPC — lighter on RaceMenu/SKEE overlay reprocessing.

Version 1.1.0

- Male bodies on/off master toggle — turn the whole male-body feature off and OBody /
  vanilla handle men. Available in the MCM (Body Shape) and the INI (MaleBodies).
- ESL-flagged plugin — no longer uses a regular load-order slot.
- Internal cleanup: source comments and log strings are English-only.

Version 1.0.0 — Initial release

Procedural NPC weight and body randomization for OBody NG, for women (CBBE 3BA) and
men (HIMBO). Highlights:

- Per-NPC procedural bodies via SKEE morphs — no preset library needed. Deterministic
  within a playthrough (same NPC always looks the same).
- Realistic majority + tunable fantasy minority.
- Body traits (independent per-NPC features) for both sexes.
- Muscle tone: athletic women (visible definition) with a rare "snu snu" Amazon tier;
  men get tone from their build. Suppressed by body fat.
- Unusual bodies: rare out-of-distribution rolls (ultra-petite/thick, ultra-skinny/huge).
- Re-roll hotkey (re-bindable, default [ / {), clothing refit, distance-aware lazy loading.
- Full MCM: distribution mode, bias, seed, morph intensity, fantasy %, unusual %,
  unusual breasts %, athletic women %, body shape mode.
- FOMOD installer with Realistic / Balanced / Fantasy starting presets + an INI for
  new-game defaults.
- All logic in the C++ SKSE plugin; Papyrus is a thin relay.
